## 快速开始

> 注意，你看到的文档是 antd-mini 1.0 alpha 文档。1.0 正式版本还没有发布。正式版本为 0.x 可访问 https://mini.ant.design

1. 安装依赖
```
$ npm i antd-mini@alpha --save
```
<br>

2. 在 json 文件中配置
``` json
{
  "usingComponents": {
    "button": "antd-mini/es/Button/index"
  }
}
```
<br>

3. 在 axml 文件中使用
```html
<button>antd-mini</button>
```
