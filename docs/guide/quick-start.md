## 快速开始

开始之前，请阅读支付宝小程序开发相关内容：[自定义组件](https://opendocs.alipay.com/mini/framework/custom-component-overview)

1. 安装依赖
``` bash
$ npm i antd-mini -S
```
<br>

2. 在 json 文件中配置
``` json
{
    "usingComponents": {
        "tag": "antd-mini/es/Tag/index"
    }
}
```
<br>

3. 在 axml 文件中使用
``` xml
<tag><tag/>
```
